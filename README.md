# Frontend-Web-Development-Jewelry-Website
💎 Baebling — Online Jewelry Website | Responsive e-commerce UI with shopping cart, product gallery, and elegant design | HTML, CSS, JavaScript


## 📌 Project Overview
**Baebling** is a visually stunning and responsive online jewelry website designed to showcase elegant jewelry collections. The website focuses on creating a luxurious and seamless shopping experience with a modern, feminine aesthetic.

## ✨ Features
- 💍 **Product Gallery** — Display jewelry collections with images, prices, and descriptions
- 🎨 **Elegant UI Design** — Soft color palette, typography, and animations
- 🔍 **Category Filters** — Browse by product type (rings, necklaces, earrings, bracelets)

## 🛠️ Technologies Used
| Technology | Purpose |
|------------|---------|
| **HTML5** | Semantic structure |
| **CSS3** | Styling, Flexbox, Grid, Animations |
| **JavaScript** | Interactivity, Cart logic, DOM manipulation |
| **Font Awesome** | Icons |
| **Google Fonts** | Typography |
